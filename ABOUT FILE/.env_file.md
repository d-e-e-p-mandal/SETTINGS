# Environment Variables — When to Use Which (All Cases)

## Plain HTML / CSS / JS
❌ No environment variables  
(values must be hardcoded or fetched from backend)

---

## React + Vite : Only Exception
**Use**
```env
VITE_API_URL=...
```
Access
```js
import.meta.env.VITE_API_URL
```

⸻

React (Create React App – old)

Use
```
REACT_APP_API_URL=...
```
Access
```
process.env.REACT_APP_API_URL
```

⸻

Next.js (Frontend)

Use
```
NEXT_PUBLIC_API_URL=...
```
Access
```
process.env.NEXT_PUBLIC_API_URL
```

⸻

Next.js (Backend / Server)

Use
```
MONGODB_URI=...
JWT_SECRET=...
```
Access
```
process.env.MONGODB_URI
```

⸻

Node.js / Express (Backend)

Use
```
MONGODB_URI=...
JWT_SECRET=...
```
Access
```
process.env.MONGODB_URI
```

⸻

One Simple Rule

Frontend → prefix required
Backend → no prefix
