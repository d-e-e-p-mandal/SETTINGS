# Built-in macOS Shortcut Settings (Recommended)

**Steps:**

1. Open System Settings
2. Go to Keyboard → Keyboard Shortcuts
3. Choose a category (e.g., App Shortcuts, Mission Control)
4. Click “+” (Add Shortcut)
5. Select:
    * App (or “All Applications”)
    * Menu title (exact name from app menu) (Mention in below)
    * Your new shortcut
6. Click Done



**Example (Custom shortcut for Copy):**

* App: All Applications
* Menu Title: Copy
* Shortcut: Cmd + Shift + C

When any app is open, you’ll see these at the top:

⸻

🔷 1. Apple Menu ()

* About This Mac
* System Settings
* App Store
* Sleep / Restart / Shut Down

⸻

🔷 2. App Name Menu

(Changes based on app, e.g., “Safari”, “TextEdit”)

* About App
* Settings / Preferences
* Quit

⸻

🔷 3. File

* New
* Open
* Save
* Save As
* Close

⸻

🔷 4. Edit

* Undo
* Redo
* Cut
* Copy
* Paste
* Delete
* Select All

⸻

🔷 5. View

* Zoom In / Out
* Full Screen
* Show / Hide elements

⸻

🔷 6. Window

* Minimize
* Zoom
* Bring All to Front

⸻

🔷 7. Help

* Search commands
* App help

⸻

🔷 Right Side (Status Menu Bar)

These are system icons:

* 🔍 Spotlight (Search)
* 📶 Wi-Fi
* 🔊 Sound
* 🔋 Battery
* 🕒 Date & Time
* 🌐 Input (Keyboard / Language)
* Control Center
